Feedback

De eerste versie van Tom's webpage was heel netjes en strak, maar met nog weinig kleur waardoor het niet de aandacht trok. Nu is zijn webpage veel kleurrijker en oogst het veel professioneler. Er is ook veel code gebruikt, vooral veel nieuwe code. Hij is al heel ver, maar de code mag nog wel wat meer gestructueerd, zodat je er makkelijk doorheen kan lezen.

Reflectie

Ik begon deze opdracht met het schrijven van een HTML webpage. Als eerste heb ik de header gemaakt. Vervolgens heb ik alle paragraven op de juiste volgorder toegevoegd aan mijn html. Daarna heb ik de afbeeldingen onder de juiste paragraven gezet. Ik vond de opdrachten best wel te doen. Ze waren niet te uitdagend, en ik was er redelijk snel mee klaar. Nadat ik klaar was met mijn HTML ben ik verder gegaan met mijn CSS. Dat ging mij best makkelijk af. Daarom ben ik gaan experimenteren met andere CSS functies, zoals gradients en keyframes. Al met al heb ik veel voor elkaar gekregen, maar had ik eerder mogen beginnen. Nu heb ik namelijk de laatste paar dagen veel moeten doen.
